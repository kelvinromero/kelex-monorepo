## Getting Started

### Prerequisites

- Docker
- Docker Compose

### Installing

Simply run the following command to build the docker image and start the container.
```sh
make setup
```

### Running the tests

```sh
make test
```

### Running the linter

```sh
make lint
```

### Running the server

```sh
make up
```
